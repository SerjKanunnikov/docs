# YandexPassportUserAccount

```json 
{
  "login": "string",
  "defaultEmail": "string"
}
```
 
Поле | Описание
--- | ---
login | **string**<br><p>Логин аккаунта Яндекс.Паспорта.</p> 
defaultEmail | **string**<br><p>Email используемый по умолчанию для пользователя с аккаунтом Яндекс.Паспорта.</p>
